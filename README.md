**Next Word**

Next word es una aplicación tanto web como movil, enfocado en ofrecer capacitaciones virtuales por profesores destacados,
La plataforma gestiona de extremo a extremo el ciclo de aprendizaje: desde la programación de horarios hasta el pago y la evaluación del servicio
en esta aplicación tiene tres roles principales :

## Características Principales
### Rol Estudiante
- **Registro y Perfil:** Gestión autónoma de cuenta.
- **Reserva Inteligente:** Selección de clases (individuales/grupales) basadas en disponibilidad real (*slots*).
- **Pagos Integrados:** Pasarela de pago para confirmación de tutorías.
- **Aula Virtual:** Acceso directo a reuniones mediante enlaces automáticos de Google Meet.
- **Feedback:** Sistema de calificación (1-5 estrellas) y comentarios privados para mejora continua.

### Rol Profesor
- **Gestión de Disponibilidad:** Herramienta para crear y administrar *slots* de tiempo sin traslapes.
- **Panel de Control:** Visualización de clases próximas y agenda organizada.
- **Comunicación:** Chat directo con alumnos vinculados a sus clases.
- **Reputación:** Perfil público basado en el promedio de estrellas otorgado por la comunidad.

### Rol Administrador 
- **Gestión de Talento:** Alta, baja y suspensión de cuentas de profesores.
- **Auditoría Financiera:** Historial completo de pagos y transacciones.
- **Monitoreo de Calidad:** Acceso a reportes de satisfacción y comentarios de alumnos.
- **Privacidad:** Supervisión operativa sin acceso a mensajes privados (Cumplimiento de privacidad).
